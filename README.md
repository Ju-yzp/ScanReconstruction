# ScanReconstruction


## 项目介绍

由作者于自身对于三维重建比较感兴趣，于是大三上从10月末开始着手看三维重建的论文以及相关源码，这个仓库是作者在做了以上这些事后，结合自己对于三维重建的理解和别人的思想，重新写的一个三维重建项目.

## 实践部分

1. 观察imu在没有实时估计零偏和系统观测误差的情况下，利用李代数进行积分，时间越长，漂移越严重
2. 利用icp迭代求解出的位姿变换，进行优化，估计imu噪声
3. 利用关键帧系统实现重定位和回环，实际得根据frame to map 才能判断是否重定位成功或者回环
4. 实现通用的多传感器软件同步，要求硬件触发是前提
